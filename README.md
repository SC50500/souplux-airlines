# Souplux Airlines!
A name badge that introduces who I am with the theme of travel because I'm traveling this summer!
[Made for FALLOUT's in-person event]

# Description
For this name badge, I kept the PCB quite simple, using a XIAO ESP32-C3 to process how many times each button on the PCB is clicked and program it to turn on. Each light represents a different destination that I'm traveling to this year after Fallout in Shenzhen.

# Sentiment
I love traveling and visiting new places, so I thought, "What better way to introduce myself than to show where I'm traveling this summer?"

# Build & Design
1. Using the tamagotchi tutorial made by @TaniWanKenobi (located on the Fallout docs & resources site), connect four switches and four LEDs to pins 1-8 on the MCU respectively.
2. Place all of the buttons and LEDs on the F.Cu and place the rest of the components (resistors, MCU, & battery) on the B.Cu
3. Add the design in the files attached & rearrange the buttons to match the PCB design.
4. And that's it! It's as simple as that ^^
